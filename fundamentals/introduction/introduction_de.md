# Einführung

Dieser Workshop bietet mehrere praktische Übungen, in denen die neuesten Verbesserungen im aktuellen Release von Oracle Database, 21c, vorgestellt werden. Unsere Mission ist es nicht nur, Ihre Daten zu verwalten, sondern Ihnen zu helfen, Mehrwert aus Ihren Daten zu ziehen. Für die anspruchsvollsten Unternehmensanwendungen und Workloads der Welt ist Oracle Database 21c eine konvergente Datenbank, die eine leistungsstarke, hochverfügbare, sichere und skalierbare Datenverwaltungsplattform für Cloud- und On-Premises-Umgebungen bereitstellt.

### Oracle Database 21c

Die 21c-Generation der konvergierten Datenbank von Oracle bietet Kunden erstklassigen Support für alle Datentypen (z. B. relationale, JSON, XML, räumliche, grafische, OLAP usw.) und branchenführende Performance, Skalierbarkeit, Verfügbarkeit und Sicherheit für alle ihre betrieblichen, analytischen und anderen gemischten Workloads.

21c ist der neueste und beste Weg, um unseren Kunden die neuesten und besten Technologien zur Verfügung zu stellen, damit sie neue Anwendungen in Betracht ziehen. Sie können jetzt beginnen und bereit für das nächste langfristige Support-Release sein. Unser aktuelles langfristiges Support-Release ist Oracle Database 19c.

Oracle Database 21c ermöglicht SQL-, REST- und API-Transaktionen über alle Arten von Datenmodellen hinweg. Es wird kurz darauf Cloud-First mit On-Premises veröffentlicht. Einige wichtige Innovationen sind:

*   Nativer binärer JSON-Datentyp
*   Blockchain-Tabellen
*   Führen Sie JavaScript-Code in der Oracle Database aus.
*   Automatisches maschinelles Lernen mit Python
*   Verbesserungen für Sharding, In-Memory-Datenbank und Diagrammanalysen.
*   APEX-Updates, einschließlich automatischer Facettensuche, Remote-Deployment mit einem Klick und mehr.

Mit 21c auf der Oracle Autonomous Database können Kunden

*   Reduzierte IT-Kosten und -Komplexität
*   Innovation freisetzen
*   Entwickeln Sie leistungsstarke, datengesteuerte Anwendungen
